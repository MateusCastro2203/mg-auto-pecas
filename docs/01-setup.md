# Fase 0 — Setup (Semana 1)

Objetivo: deixar a operação legal e pronta pra vender. Tudo isso cabe no orçamento.

## 1. Abrir o MEI

- Onde: [gov.br/empresas-e-negocios](https://www.gov.br/empresas-e-negocios/pt-br/empreendedor) (Portal do Empreendedor). Abertura **gratuita**.
- Um dos sócios abre o MEI e vira o **titular formal** da operação (o outro entra no acordo de sociedade — ver [divisão de trabalho](06-divisao-trabalho.md)).
- CNAE sugerido: **47.63-6-03 — Comércio varejista de peças e acessórios novos para veículos automotores**.
  - Pode adicionar CNAEs secundários (acessórios, etc.) sem custo.
- Custo mensal: **DAS ~R$ 76/mês** (valor fixo, atualiza por ano — confirmar o valor vigente).
- Por que MEI cedo:
  - Dá **CNPJ** → conta de vendedor ML com limites maiores
  - Permite **emitir nota fiscal** (o ML pode exigir)
  - Imposto simplificado e fixo (não sobre cada venda)
- Dá pra começar como **pessoa física (CPF)**, mas o ML limita valor de vendas e retém imposto na fonte. MEI compensa já nas primeiras semanas.

> Limite de faturamento do MEI: **R$ 81.000/ano** (~R$ 6.750/mês). Ao ultrapassar, migra pra ME. Longe da realidade no início.

## 2. Criar a conta de vendedor no Mercado Livre

- Criar conta com o **CNPJ do MEI**.
- Vincular **Mercado Pago** + conta bancária (é por onde o dinheiro cai).
- Preencher dados fiscais completos (endereço, regime tributário Simples/MEI).
- Configurar prazo de manuseio o **mais curto possível** (mesmo dia / 1 dia útil).

## 3. Ativar Mercado Envios

Duas opções principais no começo:

| Modo | O que é | Quando usar |
|------|---------|-------------|
| **Flex** | Vocês mesmos postam (Correios/transportadora via etiqueta do ML) | **Começar aqui** — pouco volume, custo baixo |
| **Full** | Estoque fica no galpão do ML; eles embalam e enviam | Depois, para os campeões de giro (ver [Fase 3](04-escala.md)) |

- No início, **Flex** com Correios via ML é mais barato e você controla tudo.
- Ative o frete grátis com estratégia (ver [precificação](05-precificacao.md)) — ele impacta ranqueamento.

## 4. Kit básico de operação

Cabe no orçamento e é suficiente pra começar:

- [ ] **Balança de cozinha digital** — calcular peso pro frete (erro de peso = prejuízo)
- [ ] **Plástico bolha, fita, caixas e envelopes** de tamanhos comuns
- [ ] **Celular com câmera boa** — fotos são metade da venda (ver [Fase 2](03-primeiras-vendas.md))
- [ ] **Impressão de etiqueta** — no começo dá pra usar lan house; depois, impressora térmica barata
- [ ] Uma **planilha simples** (ou usar o próprio painel do ML) pra controlar: produto, custo, preço, estoque, lucro por venda

## 5. Documentos e organização

- [ ] Fazer o **acordo entre os sócios por escrito** antes de vender (quem entra com quanto, como divide lucro e trabalho). Ver [divisão de trabalho](06-divisao-trabalho.md).
- [ ] Separar uma **conta bancária/Mercado Pago só do negócio** — nunca misturar com dinheiro pessoal.

## Saída da Fase 0

Ao terminar, vocês têm: MEI ativo + conta de vendedor ML configurada + Mercado Envios Flex + kit de embalagem. Pronto pra escolher produtos.
