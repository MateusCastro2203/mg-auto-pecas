# Checklist acionável — do 0 à primeira venda

Marque conforme for fazendo. Ordem sugerida.

## Semana 1 — Setup (Fase 0)

- [ ] Combinar e **escrever o acordo de sociedade** (capital, divisão de lucro, papéis, saída)
- [ ] Abrir **MEI** (CNAE 47.63-6-03 — peças e acessórios novos)
- [ ] Criar **conta de vendedor no ML** com o CNPJ
- [ ] Vincular **Mercado Pago** + conta bancária **só do negócio**
- [ ] Configurar **prazo de manuseio curto** (mesmo dia / 1 dia útil)
- [ ] Ativar **Mercado Envios Flex**
- [ ] Comprar **kit de operação**: balança, plástico bolha, caixas/envelopes
- [ ] Definir quem é o **dono do atendimento e do despacho** (não pode ser ambíguo)
- [ ] Criar **planilha de controle** (produto, custo, taxas, preço, lucro, estoque, giro)

## Semana 1-2 — Produtos (Fase 1)

- [ ] Fazer **pesquisa de demanda** no ML (Mais vendidos, unidades vendidas, perguntas)
- [ ] Escolher **3-5 SKUs** que batem os critérios (leve, barato, giro, markup 2-3x)
- [ ] Achar **fornecedor nacional** pra cada SKU
- [ ] Rodar o **simulador de tarifas do ML** pra cada produto
- [ ] Comprar o **primeiro lote pequeno** (não comprar volume de algo não testado)

## Mês 1 — Primeiras vendas (Fase 2)

- [ ] Tirar **fotos próprias boas** (fundo limpo, vários ângulos, produto em uso)
- [ ] Escrever **títulos otimizados** (marca + produto + modelo/medida + compatibilidade)
- [ ] Preencher **ficha técnica 100%** e descrição com compatibilidade correta
- [ ] Publicar no **Clássico** (giro) / **Premium** (ticket alto)
- [ ] Definir **preço competitivo** de entrada (margem menor no começo)
- [ ] **Responder perguntas em minutos**
- [ ] **Postar no mesmo dia** toda venda
- [ ] **Embalar caprichado** + bilhete de agradecimento
- [ ] Meta: **15-30 vendas** com **termômetro verde**

## Mês 2-3+ — Escala (Fase 3)

- [ ] **Reinvestir 100% do lucro**
- [ ] **Dobrar** os campeões, **cortar** os que não giraram em 30 dias
- [ ] Migrar campeões pro **Full**
- [ ] Ativar **Mercado Ads** (orçamento pequeno, só campeões)
- [ ] Identificar o **nicho/modelo campeão** e começar a aprofundar catálogo (Caminho C)
- [ ] **Subir preços** conforme reputação sobe
- [ ] Acompanhar métricas semanais (faturamento acumulado, lucro/SKU, reputação)

## Meta final

- [ ] **R$ 10.000 de faturamento acumulado** 🎯
