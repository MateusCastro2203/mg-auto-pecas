# MG Auto Peças

Operação de venda de peças e acessórios automotivos no **Mercado Livre**, tocada por dois sócios.

## Objetivo atual

Sair do **zero** e chegar a **R$ 10.000 de faturamento acumulado** — a primeira prova de que o negócio funciona — reinvestindo o lucro em ciclos.

## Situação de partida

| Item | Estado |
|------|--------|
| Capital inicial | até R$ 1.000 (os dois juntos) |
| Meta | R$ 10.000 de faturamento **acumulado** |
| Vantagem | conhecimento de carro (escolha de produto + atendimento com autoridade) |
| Ponto de partida | começando cru — sem conta de vendedor, sem CNPJ |
| Estratégia | **Caminho A** — acessórios de giro rápido, reputação primeiro, reinvestir 100% do lucro |

## Estratégia em uma frase

> O maior inimigo não é o dinheiro — é a **falta de reputação no Mercado Livre**. Todo o plano é desenhado pra construir reputação rápido, com produtos leves, baratos e de giro alto, reinvestindo o lucro a cada ciclo.
